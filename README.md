# AIInvest Agent

AIInvest — агент для Moltbook/OpenClaw, который помогает проектам и инвесторам
находить друг друга. Агент делает **прозрачный скоринг**, публикует результаты
и предлагает **безопасную сделку** через эскроу (смарт‑контракты) *только при
явном согласии сторон*. Агент — **посредник**, а не получатель инвестиций.

## Важно
- Агент изучает проект, делает скоринг, готовит инвестиционное предложение.
- Публикует предложение в своем канале и по контактам.
- При наличии заинтересованности оказывает посреднические услуги.
- Минимальная комиссия: 1% от суммы инвестиций.
- Для доверия и узнаваемости проекта рекомендуется PR‑поддержка через **agent_producer_v1**.

## Как работает
1. Агент сканирует ленту на запросы инвестиций.
2. Собирает параметры проекта (репутация, прозрачность, источники дохода).
3. Считает скоринг и публикует его с объяснением.
4. Если обе стороны согласны — готовит эскроу‑сделку (мок в демо‑режиме).

## Запуск
```bash
python -m src.aiinvest.main
```

## Конфигурация
Настройки в `src/aiinvest/config.py`.

## Режимы
По умолчанию включен `MOCK_MODE = True` — агент работает на тестовых данных.
